# SNAKE — Broken Reality Edition

Класична гра Змійка у стилі матриці з неоновим дизайном.

## Як грати

- **Стрілки / WASD** — керування змійкою
- **Пробіл** — пауза
- **Свайпи** — керування на телефоні
- Їж червону їжу, щоб рости і набирати очки
- Не врізайся у стіни, перешкоди та у себе!

## Рівні складності

| Рівень | Швидкість | Опис |
|--------|-----------|------|
| Easy   | Повільна  | Для початківців — більше часу на реакцію |
| Normal | Середня   | Класичний режим |
| Hard   | Швидка    | Для досвідчених гравців |

## Ігрові рівні

Гра має 5 рівнів, які відкриваються по мірі набору очок:
- **Рівень 1** — чисте поле
- **Рівень 2** — з'являються блоки в кутах
- **Рівень 3** — горизонтальні стіни
- **Рівень 4** — бокові стіни
- **Рівень 5** — лабіринт у центрі

## Кольори змійки

7 кольорів на вибір: зелений, блакитний, фіолетовий, помаранчевий, рожевий, золотий, білий.

## Фічі

- Неонова графіка у хакерському стилі
- Змійка з очима — голова дивиться в напрямку руху
- Частинки при з'їданні їжі
- Їжа пульсує
- Пауза на пробіл
- Свайп-керування для мобільних
- Збереження рекорду та налаштувань у браузері
- Індикатор нового рекорду
- Кнопка MENU після Game Over

## Запуск

Просто відкрий `index.html` у браузері — ніяких залежностей не потрібно!

## Технології

- HTML5 Canvas
- Vanilla JavaScript
- CSS3 з неоновими ефектами
