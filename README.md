# SNAKE — Broken Reality Edition

Класична гра Змійка у стилі матриці з неоновим дизайном.

## Як грати

- **Стрілки / WASD** — керування змійкою
- **Пробіл** — пауза
- **Свайпи** — керування на телефоні
- Їж червону їжу, щоб рости
- Не врізайся у стіни та у себе!

## Фічі

- Неонова графіка у хакерському стилі
- Змійка з очима — голова дивиться в напрямку руху
- Частинки при з'їданні їжі
- Їжа пульсує
- Пауза на пробіл
- Свайп-керування для мобільних
- Збереження рекорду в браузері
- Індикатор нового рекорду
- Швидкість зростає з кожною з'їденою їжею

## Запуск

Просто відкрий `index.html` у браузері — ніяких залежностей не потрібно!

## Технології

- HTML5 Canvas
- Vanilla JavaScript
- CSS3 з неоновими ефектами
